https://victoriasharon-stack.github.io/MineSweeper/

what it has:
1) there are 3 diff levels easy, medium and hard.
2) the first click is always safe cause the mines after placed only after you initially dig!
3) a restart feature and a high score dialogue box thingy and all the basic stuff.


How can you run it locally?
git clone https://github.com/victoriasharon-stack/MineSweeper.git
cd MineSweeper
python3 -m http.server 8000
# then open http://localhost:8000

( a local server here is needed because sometimes the browser acts like a grandma and restricts file:// saying script loading, so you can js click on index.html in the browser to double check )
